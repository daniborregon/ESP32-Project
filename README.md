Vehículo Terrestre Controlado por ESP32 - Informática Industrial
 
Repositorio oficial del Trabajo de la Asignatura de **Informática Industrial** (Grado en Ingeniería Electrónica Industrial, Curso 2025/2026).

Este proyecto consiste en el diseño y desarrollo de un sistema robótico basado en la arquitectura Dual-Core del microcontrolador ESP32, implementando un Sistema Operativo en Tiempo Real (FreeRTOS) para la gestión concurrente de tareas.
